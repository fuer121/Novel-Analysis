# Phase 5 UAT 清单

本清单只定义 UAT 准入与证据，不授权正式飞书回调变更、真实 Dify 调用或 UAT 执行

| 字段 | 必填内容 |
| --- | --- |
| Owner | UAT 组织人与 3 至 5 名管理员、成员、核心分析代表用户 |
| Approver | UAT Gate 审批人 |
| Input | 已批准快照、候选 HTTPS origin、精确飞书 callback、代表用户名单与已完成重建的测试书籍 |
| Evidence | 登录、拒绝未授权身份、书库、章节、readiness、L2 提问、证据查看、任务恢复与权限拒绝记录 |
| Hard stop | 回调不精确、用户范围未批准、阻塞问题未关闭、证据缺失或恢复路径不可用时停止 |
| Gate dependency | 正式快照 Gate、目标服务器演练 Gate 与飞书 callback Gate 均通过后申请 UAT Gate |

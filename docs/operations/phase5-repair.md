# Phase 5 切换后修复清单

本清单只定义新系统维护模式下的修复准入，不授权停止服务、删除数据库、恢复旧入口或执行修复

| 字段 | 必填内容 |
| --- | --- |
| Owner | 新系统应用、Worker、数据库与安全修复负责人 |
| Approver | 修复操作 Gate 审批人 |
| Input | 已确认故障范围、维护模式影响、最近备份指纹、修复方案、验证方案与观察负责人 |
| Evidence | 故障时间线、审批记录、维护模式记录、变更记录、恢复验证、审计与观察结果 |
| Hard stop | 根因或影响范围不明、备份不可验证、修复扩大数据或权限语义、要求恢复旧入口或缺少审批时停止 |
| Gate dependency | 正式切换后由独立修复操作 Gate 授权，旧备份 90 天到期销毁另行审批 |

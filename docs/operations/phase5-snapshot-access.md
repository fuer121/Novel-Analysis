# Phase 5 正式快照访问清单

本清单只定义后续 Gate 输入，不授权访问、复制、解密或销毁正式数据

| 字段 | 必填内容 |
| --- | --- |
| Owner | 正式快照访问执行负责人 |
| Approver | 正式快照与旧密钥访问审批人 |
| Input | 只读快照标识、来源指纹、访问起止时间、隔离存放位置与旧密钥保管人 |
| Evidence | 审批记录、来源指纹、访问审计、隔离存放证明、权限证明与销毁证据 |
| Hard stop | 缺少审批、指纹不一致、存放不隔离、权限超出只读或旧密钥保管人缺席时停止 |
| Gate dependency | 正式快照访问 Gate，且必须早于目标服务器演练与 UAT |
